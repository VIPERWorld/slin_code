Documentation

Helper class used for get and traverse all files in specified directory and subdirectories.

IMPORTANT:This class can only work under Microsoft Visual C++ platform now.

FileFinder(const std::string filter, const std::string directory = ".", bool include_subdir = true)

Parameters:
filter - a string that specify filter you can apply to the file
directory - a string that specify base directory to find, default for current directory, DO NOT end up with character '\'
include_subdir - indicate whether include subdirectories or not
